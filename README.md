# 11_ADC_Filtros
Igual que el anterior, solo que aquí se aplican (de manera muy simple) tres filtros: un promedio móvil, un IIR y un FIR.
Por compilación condicional se puede elegir que se convierta a 8, 10 o 12bits y además se habilita si se quiere usar la extensión Teleplot para observar graficamente.
